## bitcask-cpp
- A storage engine for learning

## Dependency:

* [Catch2](https://github.com/catchorg/Catch2)
* [CRC32](https://github.com/google/crc32c)
* [Zstd](https://github.com/facebook/zstd)

## How to Run
```
cmake -B build
cmake --build build

```
